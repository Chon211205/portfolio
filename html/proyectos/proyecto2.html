Calculadora Web

Calculadora enfocada en operaciones basicas con test y storybooks funcionales.

Tecnologías

TypeScript CSS JavaScript HTML

Qué demuestra

El objetivo de este proyecto es prácticar los conceptos alrededor diseño de aplicaciones en base a componentes. También aplicar testing, linting y Storybook.

Reflexión

Este proyecto me ayudo a ver como funcionan los testing, linting y Storybook. En mi opinión son útiles para proyectos grandes para comprobar si algo falla y evitar acumular errores.
